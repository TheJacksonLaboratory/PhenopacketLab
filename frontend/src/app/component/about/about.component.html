About

An application to upload, edit and save Phenopackets. Phenopackets are JSON or YAML files with a structure that follows the Phenopacket Schema standard. An open standard for sharing disease and phenotype information to improve our ability to understand, diagnose and treat both rare and common diseases. This application is still under development and allows currentlyonly the upload of JSON phenopacket files. Our current roadmap includes creation, saving, and searching of phenopackets as well as user authentication.
